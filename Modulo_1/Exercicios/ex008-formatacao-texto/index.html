Principais formatações

Negrito/ Destaques

Nesta frase temos um termo em negrito usando a <b> (não é semântica)

Nesta frase temos um termo em destaque usando a <strong> (semântica)

Itálico/ Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase, temos um termo em itálico usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande (<big> obsoleto) e um texto pequeno (<small> funcional) nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído <del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido <ins> para dar uma ênfase e indicar que ele foi adicionado depois. (semântica)

Existe também o sublinhado com tag <u> (não semântica)

Texto Sobrescrito

Para inserir coisas como x20+3 <sup>

Texto Subscrito

Para inserir coisas do tipo H2O <sub>
